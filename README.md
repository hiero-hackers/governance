# governance

Hiero Hackers governance repository.

Want to create a repository in Hiero-Hackers?

Anyone in the community can request a repository.

Just open an issue here:

➡️ https://github.com/hiero-hackers/governance/issues

Tell us:

- the repository name
- a short description
- optionally: additional committers
- A maintainer will create the repository for you and give you other committers write rights to it.
